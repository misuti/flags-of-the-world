# flags-of-the-world

Flags of the World, a dataset for other purposes.

### Getting Started

You can get started by parsing `countries.json` or `countries.min.json`

#### Structure

```json
[
	{
		"country": "Afghanistan",
		"flags": [
			{
				"width": 300,
				"height": 180,
				"path": "./flags_300x180/afghanistan.png"
			},
			{
				"width": 285,
				"height": 171,
				"path": "./flags_285x171/afghanistan.png"
			},
            ...
        ]
    },
    ...
]
```
